# Data

Place the UCI Wine Quality CSV file in this folder:

```text
data/winequality-red.csv
```

The expected target column is:

```text
quality
```

If you do not have the dataset yet, generate a demo dataset:

```bash
python src/data_utils.py --create-sample
```

The included `winequality-red.csv` file is downloaded from the UCI Machine Learning Repository:

```text
https://archive.ics.uci.edu/ml/machine-learning-databases/wine-quality/winequality-red.csv
```
